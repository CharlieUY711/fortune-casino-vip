# Comandos de PowerShell para Subir el Código a GitHub Sigue estos pasos en tu terminal de PowerShell para inicializar y subir el proyecto a `https://github.com/CharlieUY711/fortune-casino-vip`: ### 1. Preparación de la Carpeta Local Abre PowerShell y navega hasta la carpeta donde guardarás los archivos HTML que descargues del lienzo: ```powershell cd "C:\Ruta\A\Tu\Proyecto\fortune-casino-vip" ``` ### 2. Inicialización del Repositorio Git Si aún no has inicializado Git en esa carpeta: ```powershell git init ``` ### 3. Conexión con el Repositorio Remoto Vincula tu carpeta local con el repositorio que creaste en GitHub: ```powershell git remote add origin https://github.com/CharlieUY711/fortune-casino-vip.git ``` ### 4. Añadir Archivos y Primer Commit Añade todos los archivos HTML y carpetas al área de preparación: ```powershell git add . git commit -m "Initial commit: Versión TESTING de Fortune Casino con soporte multi-idioma" ``` ### 5. Subir el Código Asegúrate de que la rama principal sea `main` y sube los archivos: ```powershell git branch -M main git push -u origin main ``` --- **Nota:** Si es la primera vez que usas Git en esa máquina, es posible que te pida autenticarte con tu cuenta de GitHub.
