# Proyecto: Fortune Casino Privado ## 1. Visión General Una plataforma de casino en línea exclusiva y privada, diseñada para una experiencia de usuario profesional y fluida. El acceso está restringido y controlado por un SuperAdministrador. ## 2. Funcionalidades Principales * **Acceso Restringido:** Registro de usuarios que requiere aprobación manual del SuperAdmin. * **Sistema de Créditos:** Uso de dinero ficticio para apuestas, con gestión centralizada en el backend. * **Variedad de Juegos:** * **6 Ruletas:** Cada una con un tema y estilo visual distinto (ej: Clásica, Las Vegas, Noir, Royale, etc.). * **3 Mesas de Blackjack:** Diferentes límites de apuesta y estilos de mesa. * **3 Mesas de Poker:** Lobby para selección de mesas y juego dinámico. * **6 Slots:** Máquinas tragamonedas con diversas temáticas y mecánicas de premios. * **Panel de SuperAdmin:** Gestión de usuarios, aprobación de registros y control de balances. ## 3. Especificaciones Técnicas * **Backend:** Supabase (Base de datos, Autenticación, Real-time para mesas). * **Frontend:** Interfaz profesional, optimizada para dispositivos móviles (según solicitado). * **Despliegue:** Repositorio en Git y URL pública `fortune.core.com.uy`. ## 4. Plan de Diseño de Pantallas 1. **Lobby / Dashboard Principal:** Selección de categorías de juegos (Ruleta, Blackjack, Poker, Slots). 2. **Pantalla de Ruleta (Modelo Noir):** Siguiendo la estética de la imagen de referencia pero optimizada. 3. **Mesa de Blackjack:** Interfaz de juego limpia con controles de apuesta. 4. **Lobby de Poker:** Listado de mesas disponibles y estados. 5. **Máquina de Slots:** Interfaz vibrante con rodillos y tabla de pagos. 6. **Panel de Administración / Registro:** Flujo de aprobación de usuarios.
